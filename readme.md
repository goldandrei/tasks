# Task Management Application

## Run the server

1. Open a terminal
2. Go to the server folder: task-manager-server
3. Run the server:
   dotnet run

The server runs on:

http://localhost:5220

## Run the client

1. Open a new terminal
2. Go to the client folder: task-manager-client
3. Install dependencies if needed:
   npm install
4. Start the client:
   npm start

The client runs on:

http://localhost:4200

## DATA

Task data is stored in:
task-manager-server/Data/tasks.json
