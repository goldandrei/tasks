# tasks
.net server

Task Management Application
Run the server
Open a terminal
Go to the server folder: task-manager-server 3.
Run the server: dotnet run
The server runs on:

http://localhost:5220

Run the client
Open a new terminal
Go to the client folder: task-manager-client
Install dependencies if needed: npm install
Start the client: npm start
The client runs on:

http://localhost:4200

DATA
Task data is stored in: task-manager-server/Data/tasks.json
